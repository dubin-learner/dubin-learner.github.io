## 2. Commands
本节的重点是MATLAB的命令行功能。

MATLAB是解释型语言，特点就是所见即所得。你在命令行里敲`3*5`，回车就会显示计算结果`15`；并且是弱类型语言，即赋值时无需特别指定值的类型。
