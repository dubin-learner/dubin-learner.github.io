# 读“EDA战争：一个硅谷从林的故事”有感
> 后人哀之而不鉴之，亦使后人而复哀后人也。——《阿房宫赋》唐·杜牧

最近读了一篇比较老的文章，讲的是上个世纪EDA公司之间的真实事件，整体并不复杂。

因为一些私人恩怨，Cadence公司高管出走到友商公司Avanti，开始挖老东家墙角，并且通过不正当手段偷取源码，最终被发现遭到法律的制裁。

然后另一个友商Synopsys出手，收购了Avanti顺理成章地补全了自己的技术版图，从中渔翁得利。

只能说，太阳底下无新事。在中国因为芯片产业被制裁的大前提下，EDA行业得到了空前的发展机会。国家扶持、资本流入，催生出了一大批EDA相关的创业公司。然而EDA技术的发展不是一蹴而就的，网上总有种“速胜论”，大意是只要集中力量，什么技术难题都会被很快攻克。有些网民对于此种论调坚信不疑，因此催生出了“遥遥领先”为口头禅企业的一大批忠实粉丝，其中某些罔顾事实的言论着实让人汗颜。

时至今日，争议满满的“鸿蒙”、“盘古”之流竟然还会有很多人为之辩经，抛开事实不谈直接扣帽子的行为，已经让所有相关的技术讨论变得毫无意义。某些企业为了追求所谓的“遥遥领先”，已经把“自研”这个本意是好的词汇逐渐污名化。国外一开源，国内就自研。有些可笑又有些可悲。就算国外不开源，自然也有各种手段能让其核心源码变成我们的自研软件。

在我所在的技术领域中，已经有至少两家公司的核心技术来自国外。有点廉耻之心的，大概还学一下Synopsys走合法途径购买源代码；至于没有廉耻之心的，走的就是当年“汉芯”的路子，打磨掉原本的商标再贴上自己的，就宣称自研成功。可这些又能如何？鼓吹“弯道超车”价值观的当下，速度够快就行了，走捷径的大有人在，反正不会有人和钱过不去。

也许等这些人日后赚的盆满钵满，还会标榜自己曾经对中国的EDA领域做出了多么大的贡献。而那些见不得光的手段，大概永远不会有人知道。人在做，天在看？大概只是一句给予失败者的无聊心理安慰罢了。毕竟史书都是为了胜利者而书写，不道德的荣华富贵，和守规矩的普通平凡，大多数人也许都会倾向于前者吧，每个人的追求并不相同；但也会存在选择正确但不那么容易的路，踏踏实实地前进。我想我会由衷地敬佩这些人。

言归正传，还是讲回这个上个世纪的事件吧。

## 序幕
一九九四年三月中的一天，在凯登斯（Cadence Design Systems）的办公大楼的总裁办公室内，气氛紧张而诡异，办公室内两个人都面无表情。在桌子后面的是凯登斯的总裁兼首席执行官，约瑟夫·卡斯特罗（Joseph B. Costello）；在桌子前面的是一位来自台湾的中国人，徐建国（Gerald ’’Gerry’’ C. Hsu），徐此时是凯登斯的芯片设计部的总经理。

徐建国正将手中的辞职信递给卡斯特罗。

在数轮对话之后，卡斯特罗已知徐建国的离开是不可避免了。

“你有什么打算？”卡斯特罗问到。

“我会先去度个假。”徐建国回答到。

“你将去那儿？”卡斯特罗说道，技巧地问起。这是一个试探性的问题，既问徐的度假之地，也问徐在离开凯登斯后将来的打算。

“我将去海滩。”徐建国回答到，躲开了对将来去向的回答。

数日后，新闻发布会结束，徐建国已经正式接任成为了阿卡瑟斯（Arcsys）的总裁与首席执行官。他接过一个找他的电话，里面是他熟悉的卡斯特罗的声音：“原来，这就是你的海滩。我希望你注意着，不要被太阳晒脱了一层皮。”然后，卡斯特罗挂掉了电话。

于是这拉开了硅谷有史以来最大的一起商业机密盗窃案法庭对抗的序幕。 

> （阿凡提：Arcsys后来与另外一家公司ISS合并，改公司名称为Avanti，阿凡提是取其音译。国内中文译文多用先驱。）
