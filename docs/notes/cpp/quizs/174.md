**问题 #174** 难度：1

根据C++23标准，这个程序的输出是什么？

```cpp
#include <iostream>

void f(int& a, const int& b) {
  std::cout << b;
  a = 1;
  std::cout << b;
}

int main(){
  int x = 0;
  f(x,x);
}
```

## 答案

<details>
<summary>内容展开</summary>

程序输出：
```cpp
01
```

**正确！**

`a` 和 `b` 都引用同一个变量 `x`。

`b` 是 `const` 引用这一事实并不能保证 `b` 不会从其他地方被修改，它只保证我们不能通过 `b` 来修改它。
