**问题 #252** 难度：2

根据C++23标准，这个程序的输出是什么？

```cpp
#include <iostream>

int main() {
    int i = '3' - '2';
    std::cout << i;
}
```

## 答案

<details>
<summary>内容展开</summary>

程序输出：
```cpp
1
```

**正确！**

`'3'` 和 `'2'` 都是普通字符字面量，类型为 `char`。但我们不知道它们具体对应的值！这由实现定义：

[§[lex.ccon]¶3.1](https://timsong-cpp.github.io/cppwp/std23/lex.ccon#3.1)：

> 由单个基本c-char、简单转义序列或通用字符名称组成的c-char-sequence的字符字面量，是在字面量的关联字符编码中指定字符的编码单元值。

`operator-` 将第二个操作数的值从第一个操作数的值中减去。但当我们不知道操作数的值时，如何知道结果是什么呢？

[§[lex.charset]¶9](https://timsong-cpp.github.io/cppwp/std23/lex.charset#9)：

> 数字0（U+0030）之后的每个十进制数字字符的编码单元值，应比前一个数字的值大1。

所以，即使我们不知道 `'3'` 和 `'2'` 的实际数值，也知道它们的差值是1。重要的是，这仅适用于十进制数字，而不适用于例如普通字母。没有保证说 `'b' - 'a'` 等于1。
