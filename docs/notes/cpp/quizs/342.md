**问题 #342** 难度：1

根据C++23标准，这个程序的输出是什么？

```cpp
#include <iostream>

int main()
{
    int i = 1;
    int& j;
    j = i;
    std::cout << j;
}
```

## 答案

<details>
<summary>内容展开</summary>

程序会在编译时报错。

**正确！**

引用必须立即初始化。未初始化`j`会导致编译错误[[dcl.init.ref]¶1](https://timsong-cpp.github.io/cppwp/std23/dcl.init.ref#1)：

> 声明类型为“T的引用”（[[dcl.ref]](https://timsong-cpp.github.io/cppwp/std23/dcl.ref)）的变量应当被初始化。

您可以在 C++ Insights 或 Compiler Explorer 上进一步探索此问题！

</details>
