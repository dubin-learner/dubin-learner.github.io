# 时光机

我总是忍不住会读一些过去收藏的文章，回想当时自己在做什么，是怀着怎样的心情。

## 1.[【精选转载】北理通信男找工作的经历及心得](/life/old-biter)

收藏这篇文章的时候，大概是我考研结果尘埃落定，准备去北理工读研的时候。

当时也曾经面临着，如果考研失败就要去立刻找工作的境地，虽然一路走来磕磕绊绊，最终还是有一点幸运成功上岸。

但我知道自己读完研之后还是要找工作的，所以就无意中搜到了这篇文章，在微信里点了收藏。

过去了这么多年，当时收藏的链接已经失效，于是我重新补充了另一篇转载，并且做了备份。

这篇文章刚写的时候甚至我还在读大一。看过后有点怀念当年，年轻并且踌躇满志的状态；也有点怀念那个充满朝气的时代，似乎一切都刚刚开始。

也许，只是我们这一代老去，被时代无声地超越了吧。
